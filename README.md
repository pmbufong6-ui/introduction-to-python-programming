1. History and Founder of Python
I learned about the origin of Python programming and its creator, Guido van Rossum. Python was developed in the late 1980s and officially released in 1991. It was designed to be a simple, readable, and beginner-friendly programming language that supports multiple programming paradigms and is widely used in fields such as web development, automation, artificial intelligence, machine learning, and data science.

6. Data Structures in Python

I learned about Python data structures used for storing collections of data efficiently.

a. Lists

Lists are ordered and mutable collections that allow duplicate values.
Common operations:

* append()
* remove()
* insert()
* indexing and slicing

b. Tuples

Tuples are ordered but immutable collections.
c. Sets

Sets are unordered collections that do not allow duplicate values.
Common operations:

* add()
* remove()
* union and intersection

d. Dictionaries

Dictionaries store data in key-value pairs.
Common operations:

* accessing values using keys
* updating values
* adding and removing items
